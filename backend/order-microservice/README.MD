# Fast Food Simulator - Order Microservice

This is the microservice used for taking and listing orders.

## Planned functionalities

1. Customers, regardless if they are registered or not, can submit an order - the only condition is that they pay for it
2. Cooks can list available orders
3. Cooks can assign themselves an order
4. Cooks can set an order as ready for delivery
5. Riders can see which orders are ready for delivery
6. Riders can mark an order as delivered
